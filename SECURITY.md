# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `psalm/plugin-laravel`, please report it privately.

Email: **github@alies.dev**

Please include:

1. A description of the issue and its impact.
2. Steps to reproduce (a minimal code sample or repository is ideal).
3. Affected versions of the plugin, PHP, Psalm, and Laravel.

You will receive an acknowledgement within a few business days. Please do not disclose the issue publicly until a fix has been released.

## Supported Versions

Security fixes are provided for supported major versions of the plugin. See [README.md](README.md) for a list of supported versions.
